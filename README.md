# 🛡️ FLAGSHIP PROJECT :: **TRIVYON**

### `OBSERVE` → `ANALYSE` → `PREDICT` → `DEFEND`

![Status](https://img.shields.io/badge/STATUS-RESEARCH%20%26%20DEVELOPMENT-00ff9c?style=for-the-badge&labelColor=0d1117)
![Type](https://img.shields.io/badge/TYPE-AI%20CYBER%20DEFENCE%20ARCHITECTURE-8A2BE2?style=for-the-badge&labelColor=0d1117)

</div>

**TRIVYON** is a conceptual, modular cyber-defence architecture I am researching and building — designed around a continuous defence loop: observing systems and traffic, analysing signals for threat indicators, predicting likely attack paths, and defending through automated and human-guided response.

### 🧬 System Architecture

```
                         ┌───────────────────────┐
                         │        TRIVYON         │
                         │  (Core Orchestrator)   │
                         └───────────┬─────────────┘
                                     │
                 ┌───────────────────┼───────────────────┐
                 │                   │                   │
        ┌────────▼────────┐ ┌────────▼────────┐ ┌────────▼────────┐
        │     ARYUKEN      │ │     RENZAI      │ │    SHIVORA      │
        │  Observation &   │ │  Analysis &     │ │  Prediction &   │
        │  Data Collection │ │  Correlation    │ │  Risk Modeling  │
        └────────┬────────┘ └────────┬────────┘ └────────┬────────┘
                 │                   │                   │
                 └───────────────────┼───────────────────┘
                                     │
                          ┌──────────▼──────────┐
                          │      TAKSHIRA        │
                          │  Defence & Response  │
                          │       Engine         │
                          └──────────┬──────────┘
                                     │
                          ┌──────────▼──────────┐
                          │        SEYON          │
                          │  Reporting, Insight  │
                          │  & Feedback Loop     │
                          └───────────────────────┘
```

<br/>

<details>
<summary><b>🔍 ARYUKEN — Observation Module</b></summary>
<br/>

Responsible for the **Observe** stage of the TRIVYON loop. Conceptually designed to ingest and normalise data from network traffic, system logs, and endpoint telemetry, forming the raw signal layer that feeds the rest of the pipeline.

- Focus: data collection, sensor/log ingestion, traffic capture
- Status: **Conceptual / Early Design**

</details>

<details>
<summary><b>🧩 RENZAI — Analysis & Correlation Module</b></summary>
<br/>

Handles the **Analyse** stage — correlating observations from ARYUKEN to identify patterns, anomalies, and potential indicators of compromise.

- Focus: correlation logic, anomaly detection, signal enrichment
- Status: **Conceptual / Early Design**

</details>

<details>
<summary><b>🔮 SHIVORA — Prediction Module</b></summary>
<br/>

Drives the **Predict** stage — using analysed data to model likely attack progression and estimate risk, informing proactive defence decisions.

- Focus: predictive modeling, risk scoring, attack-path estimation
- Status: **Conceptual / Research Phase**

</details>

<details>
<summary><b>🛡️ TAKSHIRA — Defence & Response Engine</b></summary>
<br/>

Executes the **Defend** stage — translating predictions and analysis into defensive actions, alerts, and mitigation recommendations.

- Focus: automated/human-guided response, mitigation strategy
- Status: **Conceptual / Research Phase**

</details>

<details>
<summary><b>📊 SEYON — Insight & Feedback Module</b></summary>
<br/>

Closes the loop — generating reports, dashboards, and feedback that refine ARYUKEN, RENZAI, and SHIVORA over time.

- Focus: reporting, visualization, continuous feedback loop
- Status: **Conceptual / Research Phase**

</details>

## 🐺 SECONDARY PROJECT :: **LONGWEI IDS**

![ML](https://img.shields.io/badge/Machine%20Learning-Random%20Forest-00ff9c?style=for-the-badge&labelColor=0d1117)
![XAI](https://img.shields.io/badge/Explainable%20AI-XAI-8A2BE2?style=for-the-badge&labelColor=0d1117)

**LONGWEI IDS** is a machine-learning-based **Intrusion Detection System** focused on network traffic analysis.

- 🌲 **Model:** Random Forest classifier for traffic classification
- 📡 **Domain:** Network Traffic Analysis
- 🐍 **Stack:** Python, Scikit-Learn
- 🔎 **Explainability:** Incorporates Explainable AI (XAI) techniques so detections aren't a black box — aiming to make model decisions interpretable for security analysts
